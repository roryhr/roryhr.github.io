Top Ten Things I've Been Up To

In no particular order, here's a new Top Ten list.

  1. Fish Shell. It's delightful.

  2. PySpark

  3. Published research! Real-time Prediction of Intermediate-Horizon Automotive Collision Risk

  4. Monolithic repos, it's handy to have all my code in one folder in one git repo.

  6. I got the canonical 3rd Gen 13in MacBook Pro. It's great!
